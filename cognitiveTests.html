Cognitive Tests

Project idea

  • Building a site for Cognitive Tests, PhD thesis of Professor Ali Al-Zahrani, contains 9 tests:
    • Stroop Test
    • Eriksen flanker test
    • Forward Corsi block test
    • Reverse Corsi block test
    • Auditory Forward Digit Span Test
    • Auditory Backward Digit Span Test
    • Digit-symbol coding test
    • Williams Inhibition Test
    • Reasoning solving problems Test
  • Control panel section to give the admin the control of:
    • Control the exams' availabilities
    • Control of data entered by the examiner
    • Questio space to contant with students
    • View examiners' result

Development tools

Designing and Developing a website with 9 tests with different techniques, the development had in two phases:

  • Start by developing the frontend by using HTML, CSS, and Bootstrap.
  • Develop the Backend by JS, and Firebase.
  • Deployment of the website using Firebase hosting.
