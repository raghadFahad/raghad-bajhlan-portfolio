Project idea

Designing and Developing a Platform for learning Arabic for non-Arabin students, contains a lot of:

  • Exam section to measure the student's level initially and after joining the platform
  • Student section which will get access to many services to learn Arabic language such like:
    • Viedo Laibrary
    • Voice chat
    • Online meeting using Zoom platform
    • Questio space to contant with teachers
    • Guide section to learn how to use the platform
  • Control panel section to give the Teacher full control to:
    • Voice chat
    • Online meeting using Zoom platform
    • Questio space to contant with students
    • View exam result
    • Create account to new student
    • Control the availability of the exam

Development tools

Designing and Developing a Platform using a lot of features and the most challenging to building a real-time voice chat using different techniques (socket.io, real-time DB).in this project, we worked as a team of two web developers with 3 phases:

  • Start by developing the frontend by using Angular Framework and Bootstrap.
  • Develop the Backend by Node js, MySQL, Firebase real-time DB, and Socket.io.
  • Deployment of the website and API on the server and linking them together.
